# CodeAlpha Stock Trading Platform

## Overview

This project simulates a stock trading environment where users can:

- View market stocks
- Buy stocks
- Sell stocks
- Track portfolio performance
- View transaction history
- Save portfolio data using file handling

## Technologies Used

- Java
- OOP Concepts
- Collections Framework
- File Handling
- Git & GitHub

## Classes Used

1. Stock
2. User
3. Portfolio
4. Transaction
5. StockTradingSystem

## Features

✔ Market Data Display

✔ Buy Stocks

✔ Sell Stocks

✔ Portfolio Tracking

✔ Transaction History

✔ File Persistence
